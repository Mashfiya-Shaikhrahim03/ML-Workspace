# 🚀 Featured Projects

## 📘 Supervised Learning

### 🏠 House Price Model
An end-to-end regression project for predicting house prices. The project includes data preprocessing, feature engineering, model training, and performance evaluation. Multiple regression algorithms were explored, with **Linear Regression** and **Lasso Regression** used to improve prediction performance and reduce overfitting.

---

### 💳 CreditWise – Loan Approval Prediction
A complete classification project for predicting loan approval decisions. The workflow includes data cleaning, feature engineering, preprocessing, model comparison, hyperparameter tuning, and evaluation to identify the best-performing classifier.

**Algorithms Explored**
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Naive Bayes
- Decision Tree
- Random Forest
- Bagging
- Boosting
- Voting Classifier

---

### 🧬 NovaGen Model

# 📊 Unsupervised Learning

### 🛒 SmartCart – Customer Segmentation
A customer segmentation project that groups customers based on purchasing behavior. Both partition-based and hierarchical clustering techniques were explored to discover meaningful customer groups.

**Algorithms Used**
- K-Means Clustering
- Hierarchical Clustering
- Dimensionality Reduction (PCA)
- Optimal Cluster Selection
- Silhouette Score Evaluation
- Cluster Visualization & Business Insights
